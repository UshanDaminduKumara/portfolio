
# Ushan Damindu Kumara — Portfolio

This is my personal portfolio website, showcasing my projects, skills, and academic work as a Computer Science undergraduate specializing in AI engineering, agentic systems, RAG pipelines, and full‑stack development.

The current version of this portfolio was rebuilt and optimized using **Google AI Studio** and **Vibe**, converting the original Next.js project into a fully static website using only **HTML**, **Tailwind CSS**, and **JavaScript**, while keeping the exact same visual design.

## 🚀 Features
- Clean, responsive, and modern UI  
- Project highlights with descriptions  
- Contact information  
- Fully static (no frameworks required)  
- Designed visually identical to the original Next.js version  

## 🛠️ Technologies Used
- HTML  
- Tailwind CSS  
- JavaScript
